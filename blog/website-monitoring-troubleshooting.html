Website Monitoring en Troubleshooting met PRTG

Door Thomas 5 juli 2024 Website Monitoring 6 min leestijd

Door gebruik te maken van een "HTTP Advanced" sensor in PRTG, kun je effectief een website monitoren. Deze sensor controleert niet alleen de beschikbaarheid van de site, maar ook of de verwachte content daadwerkelijk wordt teruggegeven.

Configuratie van de HTTP Advanced Sensor

Volg hiervoor deze stappen:

  1. Maak een HTTP Advanced Sensor aan.
  2. Stel de sensor zo in dat deze de response controleert door te zoeken naar een specifiek woord op de opgehaalde pagina. In dit geval het woord "Features"
  3. Zie onderstaande twee screenshots voor meer details.

De status van de website wordt vervolgens automatisch elke 5 minuten gecontroleerd.

Proactieve Probleemidentificatie

Door een website actief te monitoren, kun je potentiële knelpunten identificeren. Zie bijvoorbeeld onderstaande grafiek.

Zoals je hier kunt zien, gaat de website elke avond rond dezelfde tijd voor langere periodes offline. Zonder monitoring zou je dit waarschijnlijk pas veel later hebben ontdekt. Klanten zullen je niet informeren wanneer een website niet werkt; in plaats daarvan zoeken ze gewoon naar een alternatief.

Troubleshooting met PRTG Logs

In de PRTG-log kun je de oorzaak achterhalen waarom de pagina offline is gegaan. In dit voorbeeld betreft het een interne 500-fout.

Om de oorzaak van de interne 500-fout te achterhalen, moet je de logbestanden van je webapplicatie nakijken. Dit verschilt per oplossing. In dit geval zal ik de logging moeten downloaden van de webserver waarop de website draait.

Conclusie

Door gebruik te maken van een HTTP Advanced Sensor in PRTG, kun je effectief de status en prestaties van je website monitoren. Dit stelt je in staat om snel knelpunten te identificeren en problemen, zoals interne 500-fouten veroorzaakt door cronjobs of verouderde plugins, op te lossen. Regelmatige monitoring zorgt ervoor dat je proactief kunt reageren op issues, wat de betrouwbaarheid en gebruiksvriendelijkheid van je website aanzienlijk verbetert.
